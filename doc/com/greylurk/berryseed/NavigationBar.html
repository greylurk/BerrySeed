Classpublic class NavigationBar
InheritanceNavigationBar Panel qnx.ui.core.Container

The NavigationBar class provides both the swipe down navigation stack for moving between different screens in your application, as well as a toolbar which screens can define tools for. It also displays the title of the current screen on top of the application stack.
